# LSE 晶振频率测试仪

基于 STM32F103C8T6 的 32.768KHz LSE 晶振高精度频率测量工具。

## 硬件平台

| 项目 | 规格 |
|------|------|
| **MCU** | STM32F103C8T6 (Blue Pill) |
| **参考时钟** | HSE 外部 8MHz 晶振 |
| **被测时钟** | LSE 32.768KHz 晶振 |
| **显示** | SSD1306 OLED (I2C: PB8/PB9) |
| **指示灯** | LED (PC13, 低电平亮) |

## 测量原理

```
┌─────────────────── 参考时钟链 ───────────────────┐
│  HSE 8MHz → PLL×9 → SYSCLK 72MHz               │
│                    ↓                             │
│            TIM2 @1MHz (PSC=71, 1us/tick)        │
│            [16位CNT + 溢出IRQ = 32位等效计数器]   │
└──────────────────────────────────────────────────┘

┌─────────────────── 被测时钟链 ───────────────────┐
│  LSE 32.768KHz → 预分频/32768 → RTC秒中断(1Hz)  │
│                     ↓                            │
│              每1秒触发一次测量采样                  │
└──────────────────────────────────────────────────┘

         RTC秒中断时读取TIM2时间戳
              ↓
    g_tim_diff = snap_now - snap_last    ← 无符号减自动处理溢出
              ↓
    f_LSE = 32768000000 / g_tim_diff     ← 实际频率(Hz)
    ppm   = (f_LSE - 32768) × 10^6 / 32768
```

## 精度对比

| 方案 | 分辨率 | 量化误差 |
|------|--------|---------|
| SysTick 1ms (旧版) | 1ms | ±1000 ppm |
| **TIM2 + 溢出 (新版)** | **1us** | **±1 ppm** |

精度提升约 **1000 倍**。

## 代码架构 (main.c)

```
┌─────────────────────────────────────┐
│  常量定义                            │  LSE_FREQ, TIM2_PSC, TICKS_PER_SEC
├─────────────────────────────────────┤
│  全局变量                            │  g_freq, g_ppm, g_tim_diff, g_cnt...
├─────────────────────────────────────┤
│  基础驱动                            │  Delay_ms, LED_Init, Error_Halt
├─────────────────────────────────────┤
│  外设初始化                           │  Clk_Init, Tim2_Init, Rtc_Init
├─────────────────────────────────────┤
│  中断服务                            │  TIM2_IRQHandler(溢出计数)
│                                     │  RTC_IRQHandler_Callback(核心测量)
├─────────────────────────────────────┤
│  OLED显示                            │  Display_Result(首次画框架+数字刷新)
├─────────────────────────────────────┤
│  主程序                              │  main(): 初始化→等待→循环显示
└─────────────────────────────────────┘
```

### 启动流程

```
LED_Init() → OLED_Init() → Clk_Init() → Tim2_Init()
  → NVIC(RTC) → Rtc_Init() → "Measuring..." → 主循环
      │              │
      ├─ HSE失败 → "ERROR:HSE fail!" + LED快闪
      └─ LSE失败 → "ERROR:LSE fail!" + LED快闪
```

## OLED 显示界面

```
┌──────────────────────────────────┐
│ LSE Crystal Test                 │  第1行: 标题
│ F:32.768 kHz                    │  第2行: 实测频率
│ Err:  -031 ppm                  │  第3行: ppm误差 (±20=良品)
│ T:1000015 #05                    │  第4行: TIM2 tick数 + 测量次数
└──────────────────────────────────┘
```

| 显示项 | 含义 | 理想值 |
|--------|------|--------|
| F | LSE 实测频率 (Hz) | **32768 Hz** |
| Err | 频率偏差 (ppm) | **±20 以内** |
| T | 两次RTC中断间的 us 数 | **1000000** |
