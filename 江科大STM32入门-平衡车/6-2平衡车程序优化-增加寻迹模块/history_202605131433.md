## PC15使用说明

PC15属于**备份域（Backup Domain）**引脚。虽然板上有32.768kHz晶振焊着，但由于 **LSE（低速外部振荡器）从未在代码中使能**，晶振处于完全无驱动状态，PC15用作GPIO输入**没有任何影响**。

`Track_Init()` 中已加入必要配置：
```c
PWR_BackupAccessCmd(ENABLE);    // 启用备份域访问
RCC_LSEConfig(RCC_LSE_OFF);     // 确保LSE关闭
```

| ○ ○ ○ ○ ○ | 999.0 | 脱线 | 减速找线 |
| ● ● ● ● ● | 0 | 全黑(粗线/弯道) | 直行 |

## 按键操作流程

```
┌────────────────────────────────────────────────────────┐
│  1. 开机上电                                           │
│     → 自稳，车体放平                                   │
│                                                        │
│  2. 按【K1】                                           │
│     → RunFlag=1, 平衡启动!                             │
│     → LED1亮，车直立                                   │
│                                                        │
│  3. 按【K2】                                           │
│     → TrackFlag=1, 寻迹启动!                           │
│     → OLED显示 "TRK:ON"                                │
│     → 车自动沿黑线前进                                  │
│                                                        │
│  4. 再按【K2】                                         │
│     → TrackFlag=0, 关闭寻迹                            │
│     → OLED显示 "TRK:OFF"                               │
│     → 车保持平衡但不再追踪黑线                          │
│                                                        │
│  5. 按【K1】                                           │
│     → RunFlag=0, 停止平衡                              │
│     → 自动关闭寻迹(TrackFlag=0)                        │
│     → LED1灭，车停止                                   │
└────────────────────────────────────────────────────────┘
```

**注意**：`RunFlag=0`（平衡停止）时会自动清零 `TrackFlag`，以防止下次启动平衡时意外进入寻迹模式。需要寻迹时，按顺序操作：**K1启动平衡 → K2启动寻迹**。
